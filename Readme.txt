# 🍽️ Yummy – Recipe Web App

**Yummy** is a modern, stylish and responsive recipe web app where users can browse delicious dishes, view ingredients, and follow step-by-step cooking instructions.

---

## 🌟 Features

- 🥗 Clean and attractive UI for food lovers
- 🔍 Search functionality to find recipes by name
- 📖 Detailed view for each recipe (ingredients + steps)
- 📱 Responsive design for mobile and desktop
- 🌐 Built using only frontend technologies (HTML, CSS, JS)

> A perfect project to showcase **frontend skills** and **UI/UX sense**.

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (possibly Flexbox / Grid)
- JavaScript (ES6+)
- External Recipe API or Static JSON
